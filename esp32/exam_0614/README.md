# 20240614_ESP32_EXAM_PUBLIC

## 七段顯示器注意事項
如果你用的是共陽極七段顯示器，請將變數``ISSUNOFBEACH``改為``1``(修改位置: 第33行)
```c
#define ISSUNOFBEACH 1
```

如果你用的是共陰極七段顯示器，請將變數``ISSUNOFBEACH``改為``0``(修改位置: 第33行)
```c
#define ISSUNOFBEACH 0
```
